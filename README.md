# BingoGenerator

このBingoGeneratorは、ビンゴカードを生成し、ビンゴゲームをするためのプログラムです。

## 目次
1. [機能](#機能)
2. [インストール](#インストール)
3. [使い方](#使い方)
4. [サポート](#サポート)

## 機能
ビンゴ番号のランダム生成
- ビンゴのルールに則って、1から75までのランダムな番号を生成します。
- 同じビンゴ番号は重複して生成されません。

## インストール
### 前提条件
- Java言語のソースコード(.Java)が実行できる開発環境
  - JDK 8以上が必要です。
  - デバイスは問いません。

### 手順
1. Java言語の開発環境を用意し、「BingoGenerator.java」を開きます。
2. コードを実行します。

## 使い方
1. **ゲームの起動**
  - コードを実行すると、ウィンドウが表示されます。
2. **ビンゴゲームの実行**
  - ビンゴカード上のボタンをクリックすると、該当するボタンが選択済みの状態になります。
3. **ゲームの終了**
  - ビンゴが揃うと、ビンゴであることを示すメッセージが表れます。
  - ウィンドウを閉じるとプログラムが終了します。
